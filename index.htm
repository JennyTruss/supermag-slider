SuperMAG Slider Demo

  • This page does not use Bootstrap or any other custom styles other than those in place for the slider.
  • It uses jQuery and jQuery UI.
  • The CSS and jQuery/Javascript that makes the slider work is broken out into separate files.
  • This demo is not designed to use AJAX, but can be modified to do so.
  • Styles in the header of this file is not necessary for the functionality of the slider.
  • All text and images are added manually and statically in the source code of this file.
  • The selected value is inserted into a hidden input tag.
  • input tag is visible for the purposes of this demo.
  • Documentation for the slider can be found at http://api.jqueryui.com/slider/
  • The slider will start either at 0 or wherever the input value is at. For example, the input below is set to a $50 kick in. input value="50"
